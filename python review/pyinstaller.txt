
pip install -U pyinstaller

pyinstaller -F -w -i icon.ico timer.py

-F (All in one file)
-w (removes termianl window)
-i icon.ico (adds custom icon to .exe)
timer.py (name of your name python file)

.exe file is located in the ./dist folder
